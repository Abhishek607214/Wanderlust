# рҹ“– About The Project

WanderLust is a full-stack travel accommodation platform inspired by Airbnb.

It allows users to explore destinations, create and manage listings, upload property images, leave reviews, and securely authenticate using Passport.js.

This project was built to strengthen my understanding of Full Stack Web Development using Node.js, Express.js, MongoDB, and RESTful APIs.

# рҹҡҖ Installation

## Clone the Repository

```bash
git clone https://github.com/Abhishek607214/Wanderlust.git
```

## Navigate to Project Folder

```bash
cd Wanderlust
```

## Install Dependencies

```bash
npm install
```

## Create a `.env` File

```env
ATLASDB_URL=your_mongodb_connection_string

SECRET=your_secret_key

CLOUD_NAME=your_cloud_name

CLOUD_API_KEY=your_cloudinary_api_key

CLOUD_API_SECRET=your_cloudinary_api_secret
```

## Start the Server

```bash
node app.js
```

or

```bash
nodemon app.js
```

Open your browser and visit

```
http://localhost:8080
```
